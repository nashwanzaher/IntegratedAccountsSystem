اريد# 💼 IntegratedAccSys - نظام المحاسبة المتكامل

نظام محاسبة متكامل متعدد الوحدات مصمم لتلبية احتياجات الشركات الصغيرة والمتوسطة، مع دعم الفوترة، إدارة المخزون، وحدات متعددة، الضرائب، الرسائل التلقائية، والتقارير الديناميكية.

---

## 📌 الميزات الأساسية

- 🔐 تسجيل دخول آمن وإدارة المستخدمين والصلاحيات
- 🧾 إنشاء الفواتير (مبيعات، مشتريات، مرتجعات)
- 📦 إدارة المخزون مع تواريخ انتهاء الصلاحية وأولوية التصريف
- 💲 دعم الوحدات المتعددة (كرتون، قطعة، إلخ)
- 💰 معالجة الضرائب والخصومات بدقة
- 🟢 دعم الدفع نقدًا، بشيك، بالتقسيط
- 📊 مصمم تقارير ديناميكي يدعم المعادلات والتصفية
- 📱 إرسال فواتير ورسائل عبر WhatsApp وSMS تلقائيًا
- 🌍 نظام ترخيص مقيد بالدولة
