# 📊 نظام محاسبي متكامل | Integrated Accounting System

نظام محاسبي متكامل يدعم الحسابات، المخازن، المشتريات، المبيعات، صلاحيات المستخدم، وإعدادات النظام، مصمم للعمل في بيئة متقدمة باستخدام:
A comprehensive accounting system supporting accounts, inventory, purchases, sales, user permissions, and system settings, designed to work on a modern environment using:

- .NET 8  
- Visual Studio 2022  
- SQL Server 2019  
- RDLC Reports  

---

## 🚩 الميزات الرئيسية | Key Features

- ✅ إدارة الحسابات المالية (رئيسية وفرعية)  
  Manage financial accounts (main and sub-accounts)  
- ✅ إدارة المخازن مع دعم متعدد الفروع  
  Inventory management with multi-branch support  
- ✅ المشتريات والمبيعات مع التحكم الكامل  
  Purchases and sales with full control  
- ✅ صلاحيات المستخدمين لتحديد الأدوار والصلاحيات  
  User permissions for role-based access control  
- ✅ إعدادات النظام مرنة لتخصيص العمل  
  Flexible system settings for customization  
- ✅ متعدد العملات مع دعم أسعار صرف مختلفة  
  Multi-currency support with exchange rate management  
- ✅ متعدد الوحدات (قطعة، كرتون، عبوة، إلخ)  
  Multi-unit support (piece, carton, pack, etc.)  
- ✅ تقارير RDLC ديناميكية مع تصدير PDF وExcel  
  Dynamic RDLC reports with PDF and Excel export  
- ✅ دعم متعدد الفروع مع إدارة مخزون مركزي  
  Multi-branch support with centralized inventory management  
- ✅ واجهة مستخدم سهلة وفعالة  
  User-friendly and efficient interface  

---

## 📦 متطلبات النظام | System Requirements

- .NET 8 SDK  
- Visual Studio 2022 or later  
- SQL Server 2019 (Express or Full)  
- Windows OS  
- Modern web browser (for reports or web UI if applicable)
